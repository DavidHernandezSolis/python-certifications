# 🔵 PCAP: Certified Associate in Python Programming

Este directorio contiene todo el material estructurado de preparación para la certificación oficial **PCAP (Examen PCAP-31-03)** del Python Institute.

---

## 📂 Estructura de Directorios (Propuesta)

*   `docs/`: Guías de examen de nivel asociado, checklists de repaso.
*   `modules/`: Temarios y lecciones avanzadas de OOP, paquetes y excepciones.
*   `exercises/`: Desafíos de nivel intermedio centrado en manipulación de strings y diseño de clases.
*   `simulators/`: Simulaciones de examen y test de práctica.

*   `diagrams/`: Mapas mentales de clases, interfaces, MRO y flujos de excepciones.

## 🎯 Módulos de Aprendizaje de nivel Asociado

El temario de preparación oficial para el examen PCAP se organiza en cuatro grandes áreas fundamentales:

### 📦 Sección 1: Módulos, Paquetes y PIP
*   Estructura e importación de módulos en Python (`import`, `from ... import`, `as`).
*   Variables especiales de módulo (`__name__`).
*   Construcción y distribución de paquetes propios.
*   Gestión de entornos y librerías de terceros con `pip`.

### 🔠 Sección 2: Strings, Métodos de Colecciones y Excepciones
*   Características y métodos avanzados de strings (búsqueda, reemplazo, formato).
*   Comprensión de listas, lambdas, funciones de orden superior (`map`, `filter`).
*   Jerarquía fundamental de excepciones.
*   Control robusto de errores mediante `try`, `except`, `else`, `finally`.

### 🏗️ Sección 3: Programación Orientada a Objetos (OOP)
*   Conceptos esenciales de objetos, atributos de clase frente a atributos de instancia, métodos.
*   Constructor `__init__`, variables ocultas y encapsulación (`__variable`).
*   Herencia (simple y múltiple) y Orden de Resolución de Métodos (MRO).
*   Polimorfismo e introspección de clases (`isinstance()`, `issubclass()`, `hasattr()`).

### 💾 Sección 4: Miscellaneous & File I/O
*   Iteradores y generadores (`yield`, `next()`).
*   Flujo y operaciones con archivos de texto (lectura, escritura, codificación, buffers).
*   Manejo de archivos binarios simples.
