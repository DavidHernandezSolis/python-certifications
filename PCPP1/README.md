# 🟣 PCPP1: Certified Professional in Python Programming 1

Este directorio contiene todo el material estructurado de preparación para la certificación oficial **PCPP1 (Examen PCPP-32-101)** del Python Institute.

---

## 📂 Estructura de Directorios (Propuesta)

*   `docs/`: Documentación de nivel profesional, guías de estudio PEP.
*   `modules/`: Temarios avanzados sobre metaclasses, tkinter, requests y sqlite3.
*   `exercises/`: Desafíos de nivel avanzado de patrones de diseño, comunicación HTTP e interfaces de usuario.
*   `simulators/`: Simulaciones de nivel profesional.
*   `cheatsheets/`: Hojas de referencia de metaprogramación y comandos de base de datos.
*   `diagrams/`: Arquitectura de red, patrones de diseño y flujo de widget en GUI.

---

## 🎯 Módulos de Aprendizaje de nivel Profesional

El examen PCPP1 está diseñado para validar habilidades profesionales en cinco secciones especializadas de Python:

### ⚙️ Sección 1: Programación Orientada a Objetos Avanzada
*   Subclaseado de tipos integrados, sobrecarga de operadores.
*   Metaprogramación, creación y control con metaclases.
*   Decoradores de clases y funciones.
*   Serialización de datos con `json` y `pickle`.

### 🖥️ Sección 2: Programación de Interfaces de Usuario (GUI)
*   Conceptos de GUI, widgets con `tkinter`.
*   Propagación de eventos y gestores de geometría (`pack`, `grid`, `place`).
*   Creación de aplicaciones de escritorio interactivas.

### 🌐 Sección 3: Programación de Redes y Servicios Web
*   Conceptos de HTTP y REST APIs.
*   Consumo y peticiones seguras mediante la librería `requests`.
*   Implementación básica de clientes/servidores XML-RPC.

### 🗃️ Sección 4: Procesamiento de Archivos y Bases de Datos
*   Lectura y escritura estructurada de formatos XML, CSV y JSON.
*   Integración y almacenamiento relacional utilizando la base de datos `sqlite3`.

### 📚 Sección 5: Buenas Prácticas y Convenciones PEP
*   Adherencia a estándares de estilo oficiales PEP 8, PEP 20 (Zen of Python).
*   Documentación de código mediante PEP 257 (Docstrings).
